排序算法

一、冒泡排序

原理

冒泡排序的原理是重复走访过要排序的数列，一次比较两个元素，如果它们的顺序错误，就把它们交换。

步骤

  1. 比较相邻的两个元素，如果第一个比第二个大，就交换它们。
  2. 对从第0个到n－1个元素重复做上述步骤，这时最大的数就浮到了数组最后的位置上。
  3. 对所有元素重复以上步骤，除了最后一个。
  4. 持续对越来越少的元素做上述步骤，直到没有任何一对数字需要比较。

二、选择排序

原理

每次在未排序的数列中选出最小的放在最前面。

步骤

  1. 在未排序序列中找到最小的元素，存放到排序序列的起始位置。
  2. 再从剩余未排序元素中继续寻找最小的元素，然后放到已排序序列的末尾。
  3. 以此类推，直到所有元素均排序完毕。

三、插入排序

原理

插入排序的工作原理是对于每个未排序数组，在已排序序列中从后向前扫描，找到相应位置并插入。

步骤

  1. 从第一个元素开始，该元素可以认为已经被排序；
  2. 取出下一个元素，在已经排序的元素序列中从后向前扫描，如果被扫描的元素（已排序）大于新元素，将该元素后移一位；
  3. 重复步骤2，直到找到已排序的元素小于或者等于新元素的位置，将新元素插入到该位置后；
  4. 重复上述步骤，直到整个数组变得有序。

四、归并排序

原理

归并排序是分治法最典型的一个应用，思想就是先递归分解数组，再合并数组。

步骤

  1. 先考虑合并两个有序数组，比较两个数组的最前面的数，谁小就先取谁，取了之后相应的指针久往后移一位。然后再比较，直至其中一个数组为空，最后把另一个数组的剩余部分复制过来即可；
  2. 再考虑递归分解，将数组分解成left和right，如果这两个数组内部是有序的，那么就可以用上面合并数组的方法将这两个数组合并排序。那么如何让这两个数组内部是有序的？可以再二分，直至分解出的小数组只含有一个元素时为止，此时认为该小组内部已经有序。只要合并排序两个相邻的小数组即可。

五、快速排序

原理

快速排序通常比同为$O(n\log n)$的其他算法更快，因此常被采用，同样采用了分治法的思想。

步骤

  1. 从数列中挑选一个元素作为主元（一般取第一个）；
  2. 将比主元大的元素放到右边，小于它的元素都放到左边（挖坑填数）；
  3. 再对左右区间递归执行第二步，直至各区间只有一个数。

六、堆排序

原理

堆排序在top k问题中出现比较频繁。采用二叉堆的数据结构来实现，把最大堆堆顶的最大数取出，将剩余的堆继续调整为最大堆，再次将堆顶的最大数取出，这个过程持续到剩余数只有一个时结束。

步骤

  1. 最大堆调整：将堆的末端子节点作调整，使得子节点永远小于父节点；
  2. 创建最大堆：将堆所有数据重新排序，使其成为最大堆；
  3. 堆排序：移除在第一个数据的根结点，并做最大堆调整的递归运算。

代码

七、希尔排序

原理

希尔排序，也称递减增量排序算法，实质是分组插入排序。由 Donald Shell 于1959年提出。希尔排序是非稳定排序算法，其复杂度跟步长的选择有关。

步骤

  1. 将数组放在一个表中对列分别进行插入排序；
  2. 选用更小的步长、更长的列重复上述过程，直到整个表只有一列。
